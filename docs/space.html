Get the Code

For SCSS or Javascript:

  • yarn add @sproutsocial/seeds-space
  • npm install @sproutsocial/seeds-space --save

Notes

The spacing tokens are REM values and are based on a modified fibonacci sequence. The values greater than or equal to 1 follow the fibonacci series while the decimal values below 1 are one half of the previous value.

Pixel values below are calculated based on the Space Base value set below. (default: 16px)
